int Sequence::fun1 ( )

Mnożenie dwóch macierzy.

Wzór na mnożenie dwóch macierzy \(A\) i \(B\) o wymiarach \(m\times n\) i \(n\times p\) jest następujący:

\[C_{i,j} = \sum\limits_{k=0}^{n-1} A_{i,k} * B_{k,j}\]

gdzie \(C\) to wynikowe macierz o wymiarach \(m\times p\).

◆ fun2()

int Sequence::fun2 ( )

Amplituda drgań wymuszonych.

Wzór na amplitudę drgań wymuszonych jednowymiarowej maszyny dynamicznej o masie \(m\), sprężystości \(k\) i tłumieniu \(c\) jest następujący:

\[A = \frac{F_0}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}}\]

gdzie \(\omega\) to częstotliwość drgań, a \(F_0\) to siła wymuszająca.
